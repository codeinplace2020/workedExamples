The Tower of Hanoi of is a fun (and tricky!) programming challenge that can serve as a great introduction to a topic called recursion.

You can read more about the Tower of Hanoi problem here: https://en.wikipedia.org/wiki/Tower_of_Hanoi

Write a program that solved the Tower of Hanoi problem by shifting the disks from the starting tower to one of the other two towers

You will be asked for a number of disks, and if you input an integer the puzzle will be solved!
